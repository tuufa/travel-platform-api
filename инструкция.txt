Инструкция по развертыванию системы
1. Требования к системе
Node.js (версия 16 или выше)

PostgreSQL (версия 12 или выше)

npm 

Git (для клонирования репозитория, если требуется)

2. Установка зависимостей
Клонируйте репозиторий (если проект хранится в Git):

git clone <URL_репозитория>
cd <папка_проекта>
Установите зависимости Node.js:

npm install

3. Настройка базы данных PostgreSQL
Создайте базу данных и пользователя, используя SQL-скрипт из файла database.sql:

psql -U postgres -f database.sql
Если PostgreSQL требует пароля, введите пароль суперпользователя (обычно postgres).

Убедитесь, что база данных travel_routes и пользователь travel_user созданы.

5. Запуск сервера
Запустите сервер:

npm start
Сервер будет доступен по адресу: http://localhost:3000.
